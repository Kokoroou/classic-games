# Tech Context

## Công nghệ sử dụng
-   Python 3.x
-   Pygame
-   Poetry

## Thiết lập phát triển
1.  Cài đặt Python 3.x.
2.  Cài đặt Poetry.
3.  Clone repository.
4.  Chạy `poetry install` để cài đặt dependencies.
